# Terraform Folder

## Install Terraform/Tofu
Docs: https://opentofu.org/

For macOS/Linux (with brew):
```bash
brew update
brew install opentofu@1.9.0
tofu -version
```

## Initialize Terraform
```bash
tofu init
```

## Plan Terraform
```bash
tofu plan
```

## Apply Terraform
```bash
tofu apply
```
